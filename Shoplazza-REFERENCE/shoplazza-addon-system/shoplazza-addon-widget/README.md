# Shoplazza Add-On Widget

A lightweight JavaScript widget that enables customers to add optional product add-ons (like protection plans) on Shoplazza storefronts.

## Features

- **Universal Theme Compatibility** - Works with any Shoplazza theme
- **Real-time Price Updates** - Shows updated prices when add-ons are selected
- **Cart Integration** - Seamlessly adds/removes items from cart
- **Responsive Design** - Mobile-friendly interface
- **Lightweight** - Minimal performance impact
- **Configurable** - Customizable styling and behavior

## Development

### Setup
```bash
# Install dependencies
npm install

# Start development server
npm run serve

# Build for production
npm run build

# Run tests
npm test
```
