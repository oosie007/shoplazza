## ğŸ”§ How It Works

1. **Widget Detection**: The widget automatically finds the `#shoplazza-addon-widget` anchor
2. **Configuration Loading**: Loads add-on settings for the current product
3. **UI Rendering**: Displays the add-on checkbox with pricing and description
4. **Form Interception**: Intercepts the "Add to Cart" form submission
5. **Property Injection**: Adds add-on information as line item properties
6. **Cart Integration**: Seamlessly integrates with Shoplazza's cart system
